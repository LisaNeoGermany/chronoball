# Chronoball

Chronoball is a dynamic Foundry VTT minigame module for turn-based ball competitions. Featuring two teams, strategic phases, comprehensive scoring, and an intuitive HUD, it brings exciting grid-based sports action to your virtual tabletop.

## Features

- Two-team turn-based ball game on the Foundry VTT grid
- Strategic gameplay with distinct phases (movement, passing, interception, scoring)
- Ball carrying, passing, fumbling, and interception mechanics
- Comprehensive scoring system
- Intuitive HUD and player panel
- Full localization support (English & German)

## Installation

1. Open Foundry VTT and navigate to **Add-on Modules**
2. Click **Install Module**
3. Paste the following manifest URL:
   ```
   https://github.com/TimRoesler/chronoball/releases/latest/download/module.json
   ```
4. Click **Install**

## Compatibility

- **Foundry VTT:** v13
- **Game System:** D&D 5e (dnd5e)
